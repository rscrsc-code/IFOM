1. Grundgerüst einer HTML-Seite

<!DOCTYPE html>
<html lang="de">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Seitentitel</title>
  <link href="css/bootstrap.css" rel="stylesheet" />
</head>
<body>

  <!-- Inhalt hier -->

  <script src="js/bootstrap.bundle.js"></script>
</body>
</html>

2. Navbar

<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container-fluid">

    <!-- Logo / Markenname -->
    <a class="navbar-brand" href="#">MeinLogo</a>

    <!-- Hamburger-Button (mobil) -->
    <button class="navbar-toggler" type="button"
      data-bs-toggle="collapse"
      data-bs-target="#navMenu">
      <span class="navbar-toggler-icon"></span>
    </button>

    <!-- Navigationslinks -->
    <div class="collapse navbar-collapse" id="navMenu">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item"><a class="nav-link" href="#section1">Über uns</a></li>
        <li class="nav-item"><a class="nav-link" href="#section2">Leistungen</a></li>
        <li class="nav-item"><a class="nav-link" href="#kontakt">Kontakt</a></li>
      </ul>
    </div>

  </div>
</nav>
⚠️ ms-auto = Links nach rechts schieben. navbar-expand-lg = ab 992px Desktop-Ansicht.

3. Grid – Bild und Text nebeneinander (gleiche Höhe)

<div class="container">
  <div class="row align-items-center">   <!-- align-items-center = vertikal zentriert -->

    <!-- Bild (links, halbe Breite auf md+) -->
    <div class="col-12 col-md-6">
      <img src="img/foto.jpg" class="w-100" alt="Beschreibung" />
    </div>

    <!-- Text (rechts, halbe Breite auf md+) -->
    <div class="col-12 col-md-6">
      <h2>Überschrift</h2>
      <p>Hier steht der Text neben dem Bild.</p>
    </div>

  </div>
</div>
✅ col-12 = volle Breite auf Handy | col-md-6 = halbe Breite ab Tablet

4. Grid – 3 Spalten (z.B. 3 Cards)

<div class="container">
  <div class="row">

    <div class="col-12 col-md-4">
      <!-- Inhalt Spalte 1 -->
    </div>

    <div class="col-12 col-md-4">
      <!-- Inhalt Spalte 2 -->
    </div>

    <div class="col-12 col-md-4">
      <!-- Inhalt Spalte 3 -->
    </div>

  </div>
</div>
⚠️ Bootstrap teilt in 12 Spalten. 4+4+4=12 → 3 gleiche Spalten. 6+6=12 → 2 gleiche.

<div id="meinCarousel" class="carousel slide">

      <img src="img/bild1.jpg" class="d-block w-100" alt="Slide 1" />
    </div>
    <div class="carousel-item">

8. Häufige Utility-Klassen

<!-- Abstände -->
class="mt-3"       /* margin-top */
class="mb-4"       /* margin-bottom */
class="ms-auto"    /* margin links = auto (schiebt nach rechts) */
class="px-4"       /* padding links + rechts */
class="py-5"       /* padding oben + unten */

<!-- Text -->
class="text-center"    /* zentriert */
class="text-start"     /* linksbündig */
class="text-end"       /* rechtsbündig */
class="fw-bold"        /* fett */
class="text-muted"     /* grau / gedimmt */

<!-- Bilder -->
class="w-100"          /* Breite 100% */
class="img-fluid"      /* responsives Bild */
class="rounded"        /* abgerundete Ecken */

<!-- Flexbox -->
class="d-flex"                  /* display flex */
class="justify-content-center"  /* horizontal zentrieren */
class="align-items-center"      /* vertikal zentrieren */
class="gap-3"                   /* Abstand zwischen Flex-Kindern */

<!-- Sichtbarkeit -->
class="d-none d-md-block"   /* auf Handy versteckt, ab Tablet sichtbar */
class="d-block d-md-none"   /* nur auf Handy sichtbar */

9. Section mit Hintergrundfarbe (z.B. grauer Bereich)

✅ container-fluid = volle Breite (für Hintergrund). container innen = Inhalt bleibt zentriert und begrenzt.

11. CSS-Variablen (eigene Farben)

/* In style.css ganz oben definieren: */
:root {
  --farbe-primary: #2c6e49;
  --farbe-accent:  #f4a261;
  --farbe-bg:      #f8f9fa;
  --farbe-text:    #333333;
}

/* Dann verwenden: */
h1 {
  color: var(--farbe-primary);
}
.btn-custom {
  background-color: var(--farbe-accent);
}

12. Smooth Scroll (in style.css)

html {
  scroll-behavior: smooth;
}
✅ Damit scrollen Anker-Links (href="#section1") sanft statt zu springen.
